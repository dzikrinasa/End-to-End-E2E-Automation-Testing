# End-to-End E2E Automation Testing

## Install
#### 1. Installing Nodejs
  - Install Nodejs from [here](https://nodejs.org/en/download/)

#### 2. Clone The Repository
  > `git clone https://github.com/dzikrinasa/End-to-End-E2E-Automation-Testing.git` 

#### 3. Navigate to Project Folder
  > `cd End-to-End-E2E-Automation-Testing`

#### 5. Install Dependencies
  > `npm install`

## Running the tests
#### Run the tests in headed mode (visible browser):
  > `npx playwright test --headed`

#### Run the tests in headless browsers:
  > `npx playwright test`

## Test Reports
#### Generate HTML report:
  > `npx playwright show-report`
